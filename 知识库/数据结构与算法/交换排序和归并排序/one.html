    # 冒泡排序

    • 实现思路
      • 初始时，有序区为空，全部记录为无序区。
      • 在无序区从后向前依次比较相邻记录，如逆序，则交换两个记录。
      • 每趟排序时，无序区关键字最小的记录被逐渐交换到无序区的第一位，即加入到有序区。
      • 最小的记录有如气泡一般逐渐向上漂浮至水面。
      • 如一趟排序时没有发生过交换，即没有逆序的情况，则排序提前结束。

    举例子：

    • 代码实现如下:
    void BubbleSort(ElemType L[], int n) {
    int i, j;
    bool exchange; //是否发生交换的标志
    ElemType tmp;
    for (i = 0; i < n - 1; i++) //最多n-1趟排序
    {
        exchange = false;
        for (j = n - 1; j > i; j--) //一趟冒泡排序
            if (L[j] < L[j - 1]) //逆序则交换
        {
            tmp = L[j];
            L[j] = L[j - 1];
            L[j - 1] = tmp;
            exchange = true; //发生了交换，不能结束排序
        }
        if (exchange == false)
            return; //如果没有发生交换则排序结束
    }
}

    # 快速排序(稍微有点难)

    • 每趟排序，使一个元素放入其最终位置，这个元素称为枢轴(pivot)，通常选序列的第一个元素。
    • 枢轴把整个序列划分为两个子序列。利用递归，分别对子序列重复相同过程，直至子序列长度为0或1为止。

    void QuickSort(ElemType L[], int s, int e)
{ int low = s, high = e; //本次划分的范围
  ElemType x = L[s]; //序列第一个元素作为枢轴
  while (low < high) {//内循环①从右到左查找比枢轴小的元素
    while (low < high && L[high] >= x)  high--;
    L[low] = L[high]; //将小数放在左侧小数序列中
        //内循环②从左到右查找比枢轴大或相等的元素
    while (low < high && L[low] < x)  low++;
    L[high] = L[low]; //将大数放在右侧大数序列中
  } //循环结束时low、high重合
  L[low] = x; //确定枢轴的最终存放位置
  if (s < low - 1)
    QuickSort(L, s, low-1);//对左侧小数序列进行递归划分
  if (high + 1 < e)
    QuickSort(L, high+1, e);//对右侧大数序列进行递归划分
}
